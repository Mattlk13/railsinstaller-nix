# Rails Installer OSX

Whats included:
  RVM: Ruby Version Manager
  JB:  Jewelery Box, the official RVM GUI
  SM:  Scripting Framework
  GCC: GNU Compiler Collection 

Whats next:
  - open a new Terminal window, only new windows will have rvm and ruby loaded
  - install other rubies `rvm install rbx,jruby`
  - watch screencasts:
    - RailsInstaller -- http://vimeo.com/34078037
